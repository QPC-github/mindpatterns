Readme

Use left & right button controls to paginate the carousel slides. Use TAB key to navigate through focusable elements in the slide. Activate screen reader to hear carousel status updates (e.g. 'Showing slide 2 of 8 - Video Game Slideshow').

For simplicity sake we have omitted any kind of animation when paginating slides, but most likely a real carousel would utilise some kind of scroll effect. Again, this is a timely reminder that our emphasis with these examples is accessibility concepts only.

Please read the documentation to understand how these examples were created.
