Readme

Listbox is an ARIA and JavaScript alternative to HTML's built-in select control. It allows custom HTML. A listbox can also be opened from a button or combobox.

Please read the documentation to understand how these examples were created.

Examples

Example 1: Single-Select with Auto Keyboard Selection

Keyboard users: TAB to control then use ARROW keys to change selected option.
