Readme

A combobox allows the user to enter their own value or choose a value from a list of options.

For combobox with typeahead / autocomplete behavior, please visit the autocomplete pattern.

Please read the documentation to understand how these examples were created.

Examples

Example 1: Auto-Selection

In this example, cycling through options with arrow keys will change the textbox value; ENTER key will submit.

Nintendo Switch
Nintendo 3DS
Playstation 3
Playstation 4
XBOX ONE
XBOX 360

Example 2: Manual-Selection

In this example, cycling through options with arrow keys will only highlight the option; ENTER or SPACEBAR is required to select.
