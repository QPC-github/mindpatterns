Readme

Use ENTER or SPACE key to open menu, then ARROW keys to navigate menu items.

A menu contains commands (menuitem, menuitemradio, or menuitemcheckbox) that execute JavaScript. If you require non-JavaScript fallbacks, consider using native form controls.

Please read the documentation to understand how these examples were created.
